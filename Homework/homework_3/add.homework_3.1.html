Если я правильно понимаю, ссылаясь на этот сайт: Почему HTML считает, что «chucknorris» — это цвет?,
то HTML преобразовывает заданное название цвета по формуле:
BruceWillis автоматически разделяя название на три составляющие RGB:
Bruc=b00c
ewil=e000
lis=0000 (так как тут не хватает значения кратное 3-м, автоматически подставляется 0).
Дальше четырехзначные значения сокращаются на 2 с правой стороны.
В итоге получаем код цвета #b0e000.
Такая же конвертация происходит и с заданным значение #candy
В этом случае код цвета #ca0000.
